# mikann-chat

これはmikann!enswapwarが作ったチャットサイト。  
アクセスした全員とチャットできるサイト。  
バグが多く、αバージョン

## 仕様

リロードしたりログ削除ボタンを押すとチャットが自分だけ消えます。  
チャットは保存されません。端末に一時的に保存されてリロードすると消えます  
名前をcookieで保存されていていちいち入力し直さなくて良いです。  

## 実行方法

基本的にこれは非公開にしたほうが良い、github上にパスワードを入れるため、非公開が推奨される。
このままでも実行可能、自分なりに改変してどうぞ。
管理者パスワードはprogramfile内のadminpasswordにパスワードを入れてください。
何個でも登録可、改行することでその分パスワードを増やすことができる。
github websiteのような静的サーバーでは動作不可、renderなどの動的サーバーでしか動作できない。

# Banについて

BANはrootにあるipbanlistにipを入れることでBANが可能、adminpasswordのように、改行して入力することで複数可能。
IPの取得についてはrenderなどのサーバー実行ログに入退室や発言のたびにIPが表示されているので、ここから参照して入力してください。
また管理者パスワードを入力したあとに出てくるwindowで一時的なBANが可能です。
上のテキストボックスに対象者の名前を入れて下のボタンを押すことで一時的なBANができる。
一時BANの解除は一時BANに使った下に解除用のテキストボックス、ボタンがあるのでそこで同様の手順でBANの解除ができます。
また一時BANは解除されるか、再デプロイでBANが解除されます。
